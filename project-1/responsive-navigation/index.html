On Hamburger Menus

Originally introduced from Xerox PARC, the hamburger menu has been a controversial design choice (also see this Nielsen Norman Group article).

It can be implemented into your design via a unicode character (a Chinese glyph for the trigram for heaven):

Hamburger menus should be executed as buttons and not hyperlinks to be symantically correct. Despite the default styling, you can redefine the background, borders, and appearance of a button using CSS.

Alternatively, you can use an image file or SVG icon like Google Materials ic_menu or FontAwesome's bars icon. If you want to get ambitious, there's also animated hamburger icons.
