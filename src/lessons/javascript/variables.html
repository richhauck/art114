In JavaScript, variables are ways to hold different types of data in memory.

String

Represents text enclosed in quotes

{% highlight "js" %} const name = "John Doe"; const prefix = 'Mr. '; const fullName = prefix + name; const altFullName = `${prefix} ${name}`; {% endhighlight %}

Number

{% highlight "js" %} let age = 25; let count = -2342334.234242; {% endhighlight %}

Boolean

{% highlight "js" %} let isDarkMode = true; let isRaining = 0; {% endhighlight %}

Array

{% highlight "js" %} let fruits = ["apple", "banana", "orange"]; {% endhighlight %} let rand = Math.round(Math.random() * 3) - 1; // Writing out to the page document.getElementById("header").innerText = fruits[rand];

Object

{% highlight "js" %} let person = { name: "John Doe", age: 25, isMarried: false }; {% endhighlight %}
