--- layout: default.njk title: JavaScript Functions ---

Functions are reusable blocks of code. They help with organization, encapsulation, and reducing repetition.

Function Declaration (Named Function):

{% highlight "js" %} function greet(name) { // Function definition return `Hello, ${name}!`; // Function body } console.log(greet("Alice")); // Function call {% endhighlight %}

Function expression:

{% highlight "js" %} const multiply = function(a, b) { return a * b; }; console.log(multiply(3, 5)); // Output: 15 {% endhighlight %}

Arrow Function (ES6):

{% highlight "js" %} const divide = (a, b) => {a / b}; console.log(divide(10, 2)); // Output: 5 {% endhighlight %}

Arguments

{% highlight "js" %} const sayHello = (name) => { console.log(`Hello ${name}`); } {% endhighlight %}
