JavaScript

Part 1

This class isn't intended to teach you JavaScript exclusively, but it's still essential to the course. As part of this exercise, I would highly encourage you to register and take the CodeAcademy beginner course "Learn JavaScript" (below).

At a minimum, you should have a basic understanding of:

  • How to incorporate JavaScript into your pages (where JS goes, how to write content to your page)
  • Variables - (strings, numbers, arrays, booleans)
  • Arrays and array iteration (loop)
  • Conditions (if...else statements)
  • Objects

Part 2

Find an open-source library (Lightbox, PhotoSwipe, Slick, etc.) that permits you to create a carousel/slideshow or lightbox and create a gallery of a minimum of 3-5 images describing your favorite place. Each image should be accompanied by a caption.

TIP: When choosing a library, make sure:

  1. It has clear instructions and good documentation,
  2. Has a good amount of GitHub stars (indicating it's well used and tested), and
  3. is well-designed and easy to use.

If it takes you more than 15 minutes to set up, you should probably be looking elsewhere.
